Alguns projetos.

Calculadora de IMC

A calculadora de IMC (Índice de Massa Corporal) é uma ferramenta usada para estimar a relação entre o peso e a altura de uma pessoa, ajudando a determinar se ela está com peso normal, sobrepeso ou obesidade.

Calculadora comum

Uma calculadora comum que realiza operações matemáticas básicas, como adição, subtração, multiplicação e divisão.

Gerador de números

Um gerador de números aleatórios que gera números dentro de um intervalo definido pelo usuário. O funcionamento básico a inserção de um valor mínimo e um valor máximo, e o gerador escolhe um número aleatório entre esses dois valores.

Quiz Game

Quiz game é um tipo de jogo interativo onde os jogadores respondem a uma série de perguntas em diversas categorias, como conhecimento geral, ciência, história, esportes, entretenimento, entre outras.

Cronômetro

Cronômetro é uma ferramenta usada para medir o tempo de forma precisa. O cronômetro é amplamente utilizado em esportes, experimentos científicos, controle de produtividade e outras atividades que exigem uma medição exata do tempo.

Temporizador

Temporizador é um dispositivo ou software que mede um intervalo de tempo pré-determinado e executa uma ação ou notifica o usuário quando esse tempo termina. Diferente de um cronômetro, que conta o tempo decorrido desde o início de uma atividade, o temporizador conta para trás a partir de um valor definido até chegar a zero.

ToDo List

Uma to-do list (lista de tarefas) é uma ferramenta usada para organizar, priorizar e gerenciar atividades ou tarefas que precisam ser realizadas. Ela permite que você registre tudo o que precisa fazer, categorizando tarefas e mantendo o foco em suas responsabilidades, o que ajuda a melhorar a produtividade e organização.

Sinaleira

Uma sinaleira (ou semáforo) é um dispositivo de controle de tráfego que regula o fluxo de veículos e pedestres.

Modo noturno

O botão de dark mode (modo escuro) é uma funcionalidade que permite aos usuários alternar a interface de um aplicativo ou site entre um tema claro e um tema escuro. A principal razão para implementar o modo escuro é melhorar a legibilidade em ambientes com pouca luz e reduzir o cansaço visual.

Gerador de cores

Um software gerador de cores é uma ferramenta projetada para criar e selecionar cores para uso em design gráfico, web design, artes e outros campos criativos.

ToDo List (com salvamento local)

Um to-do list com localStorage é uma aplicação de lista de tarefas que utiliza o armazenamento local do navegador (localStorage) para salvar e recuperar tarefas. Isso permite que os dados da lista sejam persistidos mesmo após o fechamento do navegador ou a reinicialização do computador, proporcionando uma experiência mais conveniente e contínua para o usuário.
